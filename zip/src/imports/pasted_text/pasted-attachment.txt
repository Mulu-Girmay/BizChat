🧠 Project Overview

Project Name: BizChat Inventory
Type: Real-Time Business Dashboard + Conversational Commerce Platform

Build a high-performance, scalable, real-time web application that allows small businesses to manage inventory, communicate with customers, and process orders directly inside a chat interface.

The frontend must integrate with a backend running at:

http://localhost:5001
